CanvasOffscreenBuffer

constructor(width, height)

Create a canvas and add it to the DOM under the element.

size([width, [height]]) : [ width, height ]

Return the actual canvas size if used without any argument.

get2DContext()

Return the context 2D of the given canvas.

toDataURL(type='image/png', encoderOptions=1)

The method returns a data URIs containing a representation of the image in the format specified by the type parameter (defaults to PNG). The returned image is in a resolution of 96 dpi.

destroy()

Free the internal resources and remove the DOM element from the tree.
